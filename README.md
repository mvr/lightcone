Lightcone
=========

Lightcone is a Game of Life search tool for conduits and oscillators,
in the style of `ptbsearch`, `catalyst` and CatForce.

Usage
-----
To build, run `git submodule update --init` and then `make`.

```
./LightCone tests/minimal.toml
```
